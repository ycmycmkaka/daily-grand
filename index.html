Daily Grand 統計分析

可直接放上 GitHub Pages，用 CSV 做歷史開獎分析
用法：你先準備一個 daily_grand_results.csv 檔，欄位建議用： date,n1,n2,n3,n4,n5,grand。
例子：2026-04-01,2,11,17,29,41,6
之後將這個 HTML 檔放去 GitHub repo，再把 CSV 一齊上載，就可以自動讀取統計。
